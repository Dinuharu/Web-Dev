My Journey to Learning HTML

Today, I began learning HTML, which is the backbone of any website. HTML stands for HyperText Markup Language. It is used to structure content on the web. I learned about basic tags like headings, paragraph, and hyperlinks, and I used them to create my first webpage!

Exploring CSS for Styling

This week, I started learning CSS, which is used to style websites. CSS allows me to change the layout, colors, and fonts of my webpages. I created a simple webpage with a header and a navigation bar, and I styled them with CSS properties like color, background-color, and padding.
